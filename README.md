### LaraGram - Laravel Instagram Clone

> git clone this_repository_url

> Run this commands:

```sh
cp .env.example .env
```

> Configure `.env` file

```sh
composer install

npm install

php artisan key:generate

php artisan migrate

npm run dev

php artisan storage:link
```
